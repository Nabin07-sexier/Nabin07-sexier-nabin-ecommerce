# 🛍 Nabin Shop – eCommerce Website

A modern and responsive online shopping website built using HTML, CSS, and JavaScript.

🔗 Live Demo: https://Nabin07-sexier.github.io/nabin-ecommerce

---

## 🚀 Features

- 🛒 Add to Cart functionality
- 🔢 Live cart counter
- 🔍 Product search bar
- 📷 Camera access button
- 💻 Responsive design (Mobile Friendly)
- 🎨 Clean and professional UI
- ⚡ Fast and lightweight frontend

---

## 🛠 Technologies Used

- HTML5  
- CSS3  
- JavaScript (Vanilla JS)
